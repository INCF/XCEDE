This example shows how FreeSurfer segmentation data can be
represented in XCEDE.  Note the terminological annotations
which link the measurements to particular term IDs used by
the various FreeSurfer components.  The term IDs can be
interpreted in the context of the nomenclature provided in
the term lists in FreeSurfer's FreeSurferColorLUT.txt,
lh.aparc.annot, rh.aparc.annot, lh.aparc.a2005s.annot,
and rh.aparc.a2005s.annot files.

The fsstats2xcede.pl script can generate an XCEDE 2 file
from the files in the stats subdirectory of a FreeSurfer
analysis.
